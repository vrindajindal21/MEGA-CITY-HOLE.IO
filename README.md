## 🎮 **Game Features**

### **Core Gameplay**
- 🕳️ **Control a growing black hole** to consume the city
- 📈 **Progressive size system** - Start small, devour skyscrapers
- ⚔️ **Hole vs hole combat** - Eliminate smaller opponents
- 🚫 **Object blocking** - Large objects resist consumption

### **Game Modes**
1. **🎯 SOLO MODE** - Consume 100% of the city in 2 minutes
2. **🏆 CLASSIC MODE** - Largest hole by time end
3. **⚔️ BATTLE MODE** - Last hole standing (no time limit)

### **Ultra-Satisfying Feedback**
- 📳 **Screen shake** on every consumption
- 🎆 **Particle burst effects** with colorful explosions
- 💫 **Dynamic flash effects** - Color-coded feedback
- 🎵 **Tier-based audio** - Different sounds per object size
- 📱 **Haptic feedback** - Mobile vibrations
- 🔥 **Growth celebrations** - "SIZE UP!", "NICE!", "GROWING!"

### **Economic System**
- 🪙 **Coin rewards** for consuming objects
- 💎 **XP progression** with level advancement
- ⬆️ **Upgrade system** - Speed, Size, Suction improvements
- 💰 **Persistent progress** - Save data across sessions

### **Visual Polish**
- 🎨 **Modern UI** with glass morphism effects
- 🌈 **Particle systems** for destruction
- 📊 **Real-time HUD** with score, coins, size tier
- 🎯 **Milestone notifications** - "TINY HOLE!" → "MASSIVE HOLE!"

## 🎯 **Getting Started**

### **Prerequisites**
- Node.js 18+ 
- npm or yarn

### **Installation**
```bash
# Clone the repository
git clone https://github.com/vrindajindal21/MEGA-CITY-HOLE.IO.git

# Navigate to project
cd MEGA-CITY-HOLE.IO

# Install dependencies
npm install

# Start development server
npm run dev
```

### **Build for Production**
```bash
# Build the project
npm run build

# Preview production build
npm run preview
```

## 🏆 **Game Mechanics**

### **Size Progression**
| Tier | Score Required | Can Consume |
|------|----------------|-------------|
| 1 | 0 | Trash, Small objects |
| 2 | 1000 | Cars, Trees |
| 3 | 2500 | Small buildings |
| 4 | 5000 | Large buildings |
| 5 | 10000 | Skyscrapers |

### **Scoring System**
- **Object mass** = Base score
- **Bot elimination** = 5000 bonus
- **Combo multipliers** = Chain consumption
- **Time bonuses** = Quick completion

### **Upgrade System**
- **Speed** - Faster movement
- **Size** - Larger starting hole
- **Suction** - Stronger pull force
